# MySQL字段约束

## 1. 概念 	

> **在创建数据库的时候，指定对数据库的列进行数据限制性的要求**



## 2. 字段约束的作用

- 保证数据的的有效性
- 保证数据的的完整性
- 保证数据的的正确性



## 3. 常见字段约束

| 约束     | 关键字      | 说明                                              |
| -------- | ----------- | ------------------------------------------------- |
| 非空约束 | not null    | 限制此列的值必须提供，不能为null                  |
| 唯一约束 | unique      | 在表中该列的值不能重复                            |
| 主键约束 | primary key | 非空约束+唯一约束，能够唯一标识数据表中的一条数据 |
| 外键约束 | foreign key | 建立不同表之间的关联关系                          |
